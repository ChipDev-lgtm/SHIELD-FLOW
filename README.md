# SHIELD-FLOW
SHIELD FLOW is a security-focused application featuring a malware detection system, email scanning, real-time alerts, and secure login with password hashing. It ensures your data and devices remain protected with advanced, reliable safeguards.

---
## Features

- **Malware Detection**: Scan files and folders for potential threats using our enhanced malware detection
- **Gmail Monitoring**: Checks your registered email for security alerts

### Tech Stack:
- Python
- SQLite
- Tkinter
